| Version | Supported |
|----------|------------|
| 1.x      | Yes |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Steps:
1. Create a private security advisory if applicable
2. Provide detailed reproduction steps
3. Include screenshots or logs if available
4. Allow reasonable time for remediation before public disclosure

---

## Security Goals

This project aims to:
- Protect sensitive ticket information
- Prevent unauthorized API access
- Improve SLA monitoring reliability
- Follow secure coding practices

---

# User Stories

| Story ID | User Story | Acceptance Criteria | Priority |
|----------|-----------|---------------------|----------|
| US-001 | As an IT Service Manager, I want to view a real-time dashboard so that I can monitor SLA risks | Dashboard updates every 2 seconds | High |
| US-002 | As a Service Desk Agent, I want to log tickets so that incidents are recorded | Ticket is saved successfully | High |
| US-003 | As a system, I want to calculate SLA so that risk is tracked | SLA updates correctly | High |
| US-004 | As a system, I want to recommend engineers so that workload is balanced | Engineer assigned based on workload | High |
| US-005 | As an Engineer, I want to view assigned tasks so that I know what to resolve | Tasks display correctly | Medium |
| US-006 | As an Engineer, I want to update tickets so that progress is tracked | Status updates saved | Medium |
| US-007 | As a Manager, I want to generate reports so that I can analyze performance | Report displays correctly | Medium |
| US-008 | As a Manager, I want to simulate crisis scenarios so that I can test system resilience | Risk levels increase | High |
| US-009 | As a user, I want alerts for SLA breaches so that I can act quickly | Alerts display instantly | High |
| US-010 | As an admin, I want secure access so that system data is protected | Unauthorized users blocked | High |

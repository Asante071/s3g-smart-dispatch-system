# Kanban Board Explanation

A Kanban board is a visual project management tool used to track tasks across different workflow stages.

In this project, columns such as To Do, In Progress, Testing, and Done represent task progression.

The board visualizes workflow by showing how tasks move between stages, helping identify bottlenecks.

Work-in-progress is controlled by limiting tasks in active stages, ensuring efficiency and focus.

This supports Agile principles such as continuous delivery, adaptability, and transparency.
